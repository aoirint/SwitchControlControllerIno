# SwitchControlControllerIno


## Requirements
- ArduinoJson
- ArduinoSTL

[arduino:avr@1.8.3でArduinoSTLがコンパイルできない問題](https://github.com/mike-matera/ArduinoSTL/issues/56)があるので、coreにはarduino:avr@1.8.2を使用する。

```bash
arduino-cli core install arduino:avr@1.8.2
```
